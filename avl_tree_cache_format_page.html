Format of AVL tree cache.

Here is described binary format in which AVL trees are stored on disk. Comprehensive theoretical information about AVL trees can be found here.

  • AvlTree structure
  • If tree is empty
    • zeroed AvlNode structure
  • Else
    • root node (AvlNode structure)
    • ... Depth-first search (DFS) algorithm starting from the left child node ...
      If node does not have left child then empty node will be written (empty AvlNode structure)
      If node does not have right child then empty node will be written (empty AvlNode structure)
      If previous two points were true then take parent node and check his right child
      If right child exists then write it and repeat algorithm for this child from the beginning, otherwise take parent node and check his right child
      Algorithm will stop if there is no parent node
Note:
Each node is written or read by WriteKey, WriteValue and ReadKey, ReadValue functions.
