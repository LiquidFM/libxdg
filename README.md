Xdg library
===========

[Documentation](http://vilkov.github.com/libxdg) - online documentation generated by doxygen.

Installation
------------

Clone/Tarball:

```bash
$ mkdir debug
$ cd debug 
$ cmake -G "Unix Makefiles" -D CMAKE_BUILD_TYPE:STRING=Release -D CMAKE_INSTALL_PREFIX:STRING=/home/user ../
$ make && make doc && make install
```
